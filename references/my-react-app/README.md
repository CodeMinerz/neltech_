# My React App

This project is a React application that manages groups and roles within a PM (Project Management) section. 

## Features

- **Groups Management**: A dedicated page for managing groups, including viewing and editing group details.
- **Roles Management**: A dedicated page for managing roles, including viewing and editing role details.
- **Data Table**: A reusable component for displaying tabular data across different pages.
- **TypeScript**: The project is built with TypeScript for better type safety and developer experience.

## File Structure

- `src/pages/PM/common/BaseIndex.tsx`: Base component for shared logic and layout.
- `src/pages/PM/groups/index.tsx`: Component for managing groups.
- `src/pages/PM/roles/index.tsx`: Component for managing roles.
- `src/components/PM/data-table/index.tsx`: Component for displaying data in a table format.
- `src/types/index.ts`: Type definitions used throughout the project.

## Installation

To install the project dependencies, run:

```bash
npm install
```

## Usage

To start the development server, run:

```bash
npm start
```
